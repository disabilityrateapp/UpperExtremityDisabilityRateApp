günlük pratikte kullanılabilecek bir hale getirdik.

Program ile ilgili bugları, diğer sorunları ve önerilerinizi ekran görüntüleri ile beraber emreozmen@istanbul.edu.tr adresinden bana iletebilirsiniz.
